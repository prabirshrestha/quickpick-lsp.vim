# quickpick-lsp.vim
[vim-lsp](https://github.com/prabirshrestha/vim-lsp) picker for [quickpick.vim](https://github.com/prabirshrestha/quickpick.vim).

# Install

```vim
Plug 'prabirshrestha/quickpick.vim'
Plug 'prabirshrestha/quickpick-lsp.vim'
```

# Usage

```vim
:PLspDocumentSymbol
:PLspWorkspaceSymbol
```
